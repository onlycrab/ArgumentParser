This is a example of using argument converter.
Demonstrator can convert values to <bool>, <int> or <arr>.
For getting array pass elements in one line with the delimiter ','.
See the class /com/github/onlycrab/argumentParser/arguments/converter/ValueConverter for a description of the converter.
Available arguments : v(value), t(type).
Try this:
  -v true -t bool
  -v off -t bool
  --value elem1,elem2,elem3 -t arr
